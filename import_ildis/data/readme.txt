Place raw data in this subdirectory

1. Download the DwCA data from http://catalogueoflife.org/DCA_Export/. Select Fabaceae in the form to get only Fabaceae.
2. Uncompress the contents of the download to this directory.
3. The only file needed is 'taxa.txt', you may delete the rest
4. File should be tab-delimitted plain text. Alter the appropriate import parameters in import_ildis/params.inc if this is not the case.
